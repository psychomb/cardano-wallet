Description

This module allows the wallet to retrieve blocks from a known Jormungandr node. This is done by providing a NetworkLayer with some logic building on top of an underlying JormungandrClient HTTP client.

It also provides facilities for starting the jormungandr node backend process. The Jormungandr NetworkLayer implementation uses the HTTP REST API of this backend.

Description

This module provides the main wallet server function for the Jörmungandr backend.

The Cardano.Wallet.Jormungandr.Network module handles synchronization with the jormungandr process.

The block format decoders are in Cardano.Wallet.Jormungandr.Network.

Functionality specific to this backend for creating transactions is in Cardano.Wallet.Jormungandr.Transaction.

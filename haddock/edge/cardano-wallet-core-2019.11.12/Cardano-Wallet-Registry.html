newWorker :: forall key resource ctx. (HasLogger ctx, HasWorkerCtx resource ctx, ToText key) => ctx -> key -> MkWorker key resource ctx -> IO (Maybe (Worker key resource)) Source #

Create a new worker for a given key. Workers maintain an acquired resource. They expect a task as argument and will terminate as soon as their task is over; so in practice, we provide a never-ending task that keeps the worker alive forever.

Returns Nothing if the worker fails to acquire the necessary resource or terminate unexpectedly before entering its main action.

Description

This module provides the main wallet server function for the Byron-rewrite Haskell node backend.

The Cardano.Wallet.Byron.Network uses the mini-protocols (ChainSync and TxSubmission) to talk with a core node and synchronize with the network.

Functionality specific to this backend for creating transactions is in Cardano.Wallet.Byron.Transaction

withPathParam :: Int -> (Text -> Text) -> (Method, Text) -> (Method, Text) Source #

Modifies the value of a path parameter at position n (indexed from 0) with the given update function. Throws if the given endpoint has no path parameter in the given position.

( "GET", "v2/wallets/2512a00e9653fe49a44a5886202e24d77eeb998fsuffix" )
